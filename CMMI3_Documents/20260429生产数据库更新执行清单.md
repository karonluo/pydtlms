## 5. 数据抽样核验 SQL

### 5.1 核验个人陈述历史回填情况

```sql
SELECT
    application_id,
    LEFT(COALESCE(growth_experience_text, ''), 40) AS growth_preview,
    LEFT(COALESCE(program_application_reason_text, ''), 40) AS reason_preview,
    LEFT(COALESCE(career_plan_text, ''), 40) AS career_preview,
    supporting_material_attachment_url
FROM dtlms_portal_application_personal_statements
ORDER BY application_id DESC
LIMIT 20;
```

核验点：

- 旧数据存在时，新字段应尽量被回填。
- `supporting_material_attachment_url` 可从 `material_list_attachment` 补到位。

### 5.2 核验个人陈述汇总文本未被误清空

```sql
SELECT
    application_id,
    LEFT(COALESCE(personal_statement_text, ''), 120) AS personal_statement_preview
FROM dtlms_portal_application_personal_statements
ORDER BY application_id DESC
LIMIT 20;
```

核验点：

- 原本已有 `personal_statement_text` 的记录不应被错误覆盖为空。
- 原本为空但三段主题字段已有值的记录，应被自动拼装成汇总文本。
